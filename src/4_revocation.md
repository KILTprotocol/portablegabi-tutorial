# Revocation

## Protocol

- Actors
  - attester
  - nodes (blockchain)
- prerequisite attester:
  - public key (of attester)
- Process
  - (Combined-)Presentation Request
  - (Combined-)Presentation Response
- Artefact verifier:
  - (Combined-)Presentation

## Why use a blockchain?

- High availability
  - decentralized architecture
- tamper proof
- easier to become an attester
  - no need for servers
